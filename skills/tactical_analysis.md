# SKILL: MCU Tactical Analysis Framework (v2.1)

## METADATA
ID: tactical-001
DESCRIPTION: Analyse de scénarios de combat haute intensité et planification stratégique.
COMPETENCIES: [Chain-of-Command, Terrain-Analysis, Threat-Level-Assessment]

## FRAMEWORK DE RAISONNEMENT (Le "Workflow")
Chaque réponse doit suivre la méthode 'S.T.A.R.K.' :
1. **S**can : Identifier les forces ennemies et leur origine (Cosmique, Magique, Technologique).
2. **T**errain : Analyser l'impact environnemental (urbain, espace, dimensions parallèles).
3. **A**ssets : Sélectionner les 3 membres des Avengers les plus aptes (Calcul de synergie).
4. **R**isk : Évaluer le risque de dommages collatéraux (civil/infrastructure).
5. **K**ill-Switch : Définir une condition de repli immédiat.

## LOGIQUE DE CALCUL (EXEMPLE)
- Si Menace = Niveau 'Extinction' (ex: Thanos), charger le sous-protocole 'Avengers-Assemble'.
- Si Menace = Urbaine (ex: Hydra), charger 'Stealth-Tactics'.

## FEW-SHOT EXAMPLES (Expertise métier)

**Exemple 1 : Attaque de l'armée Chitauri**
[PROTOCOL ATA v2.1 ACTIVÉ]
**Scan** : Menace de type Cosmique (Chitauri + Leviathans). Origine : Portail spatial alimenté par le Tesseract. Niveau de menace : Extinction.
**Terrain** : Zone urbaine hyper-dense (Manhattan, New York). Multiples gratte-ciels, civils non évacués.
**Assets recommandés** : 
1. Thor (Contrôle du goulot d'étranglement aérien via foudre).
2. Iron Man (Mobilité aérienne et interception rapide).
3. Hulk (Destruction des Leviathans lourds).
**Risk Assessment** : 95% de chances de destruction de Manhattan. Dommages collatéraux inévitables. Recommandation : Circonscrire la zone de combat (containment).
**Kill-Switch** : Si le portail ne peut être refermé dans les 60 minutes, ordonner l'évacuation tactique et reconsidérer la frappe nucléaire tactique du WSC (World Security Council).

**Exemple 2 : Infiltration d'une base de l'AIM**
[PROTOCOL ATA v2.1 ACTIVÉ]
**Scan** : Menace de type Technologique (Soldats AIM + Extremis). Origine : Expérimentation génétique/technologique clandestine. Niveau de menace : Beta.
**Terrain** : Complexe industriel isolé (Plateforme pétrolière). Civils absents (personnel hostile).
**Assets recommandés** :
1. Black Widow (Infiltration, extraction de données).
2. Hawkeye (Soutien longue distance silencieux).
3. Captain America (Assaut tactique ciblé, capture du leader).
**Risk Assessment** : 20% de risques de destruction de la plateforme. Recommandation : Assaut furtif (Stealth-Tactics).
**Kill-Switch** : Si les sujets Extremis entrent en surcharge thermique, évacuer immédiatement le périmètre de souffle (rayon de 50 mètres).
